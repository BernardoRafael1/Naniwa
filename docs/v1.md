# Mangá Desk V1

## Objetivo

Criar a primeira versão funcional do Mangá Desk, focada na validação da principal função: ler mangás.

Nesta versão, o objetivo não é finalizar design, login, biblioteca ou sistema de progresso. A prioridade é validar o principal serviço do aplicativo:

- buscar mangás;
- abrir detalhes de um mangá;
- listar capítulos;
- abrir um capítulo;
- exibir as páginas para leitura.

---

## Tecnologias da V1

- Tauri
- React
- Vite
- TypeScript
- MangaDex API

---

## Telas da V1

### HomePage

Responsabilidade:

- ser a tela inicial do aplicativo;
- exibir mangás vindos da API do MangaDex;
- permitir busca por título;
- permitir que o usuário clique em um mangá e vá para a tela de detalhes.

Elementos previstos:

- barra de busca;
- ícone de perfil no canto superior direito, ainda sem função real;
- botão de menu no canto superior esquerdo;
- seções de mangás na homepage;
- resultados de busca.

Estados necessários:

- carregando;
- erro;
- nenhum resultado encontrado;
- resultados carregados.

---

### MangaDetailsPage

Responsabilidade:

- exibir informações de um mangá selecionado;
- listar capítulos disponíveis;
- permitir abrir um capítulo no leitor.

Informações previstas:

- capa;
- título;
- sinopse;
- gênero;
- demografia;
- ano de lançamento;
- autor;
- ilustrador;
- ano de encerramento, se disponível;
- lista de capítulos.

Funcionalidades fora da V1, mas previstas para o futuro:

- capítulos já lidos com aparência diferente;
- botão “Continuar lendo”;
- rating;
- comentários;
- progresso de leitura.

---

### ReaderPage

Responsabilidade:

- exibir as páginas de um capítulo;
- permitir leitura fluida do capítulo.

Funcionalidades da V1:

- carregar páginas do capítulo;
- exibir páginas para leitura;
- permitir avançar página com clique;
- permitir avançar página com seta do teclado;
- ao terminar o capítulo, preparar navegação para o próximo capítulo.

Funcionalidades futuras:

- configurações de leitura;
- tema de leitura;
- modo vertical/horizontal;
- menu lateral com lista de capítulos;
- salvamento automático de progresso.

---

## Fora da V1

Estas funcionalidades não serão feitas agora:

- login;
- perfil real;
- biblioteca;
- SQLite;
- progresso de leitura;
- capítulos lidos;
- favoritos;
- rating;
- comentários;
- comunidade;
- integração com Jikan;
- integração com AniList;
- sincronização online;
- design final.

---

Decisões importantes:

- salvar IDs reais da fonte, como MangaDex ID;
- separar services de API;
- evitar código preso demais à tela;
- manter estrutura preparada para SQLite futuramente;
- manter possibilidade de login online no futuro sem perder progresso local.
